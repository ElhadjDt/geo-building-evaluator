# Geo Building Evaluator

A local tool to compare predicted building data against ground truth and evaluate classification accuracy.

## Install & Run
```bash
git clone https://github.com/ElhadjDt/geo-building-evaluator.git
cd geo-building-evaluator
```
```bash
npm install
npm run dev
```

Open **http://localhost:3002** in your browser.
